## 📋 ΒΗΜΑΤΑ ΓΙΑ ΝΑ ΛΕΙΤΟΥΡΓΗΣΕΙ ΤΟ APP

### 🔴 1. ΡΥΘΜΙΣΗ SUPABASE (ΥΠΟΧΡΕΩΤΙΚΟ)

#### Βήμα 1.1: Δημιουργία Supabase Account
1. Πήγαινε στο **https://supabase.com**
2. Κάνε **Sign Up** (δωρεάν)
3. Κάνε **Create New Project**
4. Όνομα: `sports-match-app`
5. Database Password: Διάλεξε ένα δυνατό password (γράψτο!)
6. Region: Επίλεξε `West EU` (για Ελλάδα)
7. Κάνε **Create new project** (παίρνει ~2 λεπτά)

#### Βήμα 1.2: Ρύθμιση Database
1. Στο Supabase Dashboard → **SQL Editor**
2. Πάτα **New Query**
3. Άνοιξε το `supabase-schema.sql` από το project
4. Αντιγράψε **όλο** το περιεχόμενο
5. Επικόλλησε στο SQL Editor
6. Πάτα **Run** (ή F5)
7. ✅ Αν δεις "Success", το database είναι έτοιμο!

#### Βήμα 1.3: Πάρε τα API Keys
1. Στο Supabase Dashboard → **Settings** (γρανάζι)
2. Πάτα **API**
3. Βρες αυτά τα 2:
   - **Project URL**: `https://xxxxx.supabase.co`
   - **anon public key**: `eyJhbGci...` (μακρύ key)

#### Βήμα 1.4: Δημιούργησε .env File
1. Άνοιξε Terminal:
```bash
cd "/Users/dimitriostsalamanios/Desktop/x1,1"
touch .env
```

2. Άνοιξε το `.env` file και γράψε:
```
EXPO_PUBLIC_SUPABASE_URL=https://your-project-url.supabase.co
EXPO_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here
```

3. Αντικατέστησε:
   - `your-project-url` με το πραγματικό Project URL από το Supabase
   - `your-anon-key-here` με το πραγματικό anon key

**Παράδειγμα:**
```
EXPO_PUBLIC_SUPABASE_URL=https://abcdefghijklmnop.supabase.co
EXPO_PUBLIC_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZSIsInJlZiI6ImFiY2RlZmdoaWprbG1ub3AiLCJyb2xlIjoiYW5vbiIsImlhdCI6MTYxNjIzOTAyMiwiZXhwIjoxOTMxODE1MDIyfQ.xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

⚠️ **ΣΗΜΑΝΤΙΚΟ:** Χρησιμοποίησε τα **ΠΡΑΓΜΑΤΙΚΑ** από το Supabase Dashboard, όχι το παράδειγμα!

### 🟡 3. ΔΗΜΙΟΥΡΓΙΑ ASSETS (ΕΠΙΘΥΜΗΤΟ - ΓΙΑ PRODUCTION)

Το `app.json` αναφέρεται σε assets που χρειάζονται για το icon και splash screen.

**Για Development (να τρέξει το app):** **ΔΕΝ χρειάζεται τώρα**

**Για Production Build:** Χρειάζεται να δημιουργήσεις:

Δημιούργησε folder `assets/` στο root:
```bash
mkdir assets
```

Χρειάζονται:
- `icon.png` (1024x1024) - App icon
- `splash.png` (1284x2778) - Splash screen
- `adaptive-icon.png` (1024x1024) - Android icon
- `favicon.png` (48x48) - Web favicon

**Σημείωση:** Μπορείς να δοκιμάσεις το app χωρίς αυτά, αλλά θα δεις warnings.

### ✅ 4. ΕΚΚΙΝΗΣΗ ΤΟΥ APP

Μετά τα βήματα 1 και 2, μπορείς να τρέξεις το app:

```bash
npm start
```

Θα ανοίξει το Expo Developer Tools. Μετά μπορείς:

**Για iOS (Mac):**
```bash
npm run ios
```
Ή σκανάρε QR code με **Expo Go** app (iOS)

**Για Android:**
```bash
npm run android
```
Ή σκανάρε QR code με **Expo Go** app (Android)

**Για Web (testing):**
Πάτα `w` στο Expo Developer Tools

## ⚠️ ΠΡΟΒΛΗΜΑΤΑ ΚΑΙ ΛΥΣΕΙΣ

### "Invalid API key"
- ✅ Έλεγξε το `.env` file - τα credentials είναι σωστά;
- ✅ Κάνε restart το app μετά το .env update

### "relation does not exist"
- ✅ Έτρεξες το `supabase-schema.sql` στο SQL Editor;

### "Row Level Security policy violation"
- ✅ Κάνε login πρώτα - οι RLS policies απαιτούν authentication

### "Cannot find module"
- ✅ Τρέξε `npm install` ξανά

### App δεν τρέχει / Expo errors
- ✅ Βεβαιώσου ότι έχεις Node.js v18+ εγκατεστημένο
- ✅ Τρέξε `npm install` ξανά
- ✅ Καθάρισε cache: `npx expo start --clear`
